# JavaScript Programming Language


## Declaring Variables

### `let`
```javascript
let total = 12.00;

// declaring multiples variables
let price = 2.12,
	name = 'Augusto',
	alien = false; 
```

### `var`
**WARNING -** When misused, it does not show errors like in the example below:

```javascript
showMessage(price);
var price = 25
```

## `const`  - Constants

```javascript
// declaring a constant
const NUMBER_MAX = 12;
```
